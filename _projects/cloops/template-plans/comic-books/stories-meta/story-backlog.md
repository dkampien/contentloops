# Bible Stories Backlog

**Purpose:** Reference list of story ideas for comic book format
**Target:** SoulStream in-app content
**Date:** 2025-11-12

---

## How to Use This Document

This is a **reference backlog** - not a production plan:
- Pick stories from this list (or add new ideas)
- Create story files using the template workflow
- **Naming convention:** Use numbered format `XX-story-name.md` (e.g., `05-moses-red-sea.md`)
- Add completed stories to `stories-index.md`
- Mark completed stories in this backlog with ✅ and story number
- This backlog can grow infinitely without cluttering the index

---

## Selection Criteria

Stories selected based on:
- **Visual appeal** - Strong visual moments for comic format
- **Popularity** - Well-known, frequently taught stories
- **Drama/Action** - Clear narrative arc with tension and resolution
- **Child-appropriate** - Suitable for children's book illustration style
- **Page fit** - Can be told effectively in 3-4 comic pages
